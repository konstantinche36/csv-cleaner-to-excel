# Задача: CSV Cleaner to Excel

## Цель
Обработать CSV-файл с заявками, нормализовать данные
и сохранить корректные записи в Excel.

## Входные данные
- CSV-файл
- колонки: date, name, phone, city, amount, status

## Выходные данные
- result.xlsx — корректные записи
- errors.csv — строки с ошибками
